## 01.install

**install nodejs**

访问 https://nodejs.org/zh-cn/ 下载并安装

```bash
# node -v
v14.17.4

# npm -v
6.14.14
```

**configure nodejs**

设置淘宝源而非 cnpm

```bash
npm install --registry=https://registry.npm.taobao.org

npm rebuild node-sass
```

**install typescript**

```bash
# sudo npm install -g typescript
# tsc -v
Version 4.3.5

# sudo npm install -g tslint
# tslint -v
6.1.3
```
